# 깃 설치와 실행
## 1. 깃 홈페이지와 다운로드
**홈페이지**
> * https://git-scm.com/

**Git 내려 받기**
> * https://git-scm.com/downloads
>   > * Git-#.##.#-64-bit.exe

## 2. 깃 설치
**깃 설치시 주요 설정**
> * 편집기 설정: Visual Studio code
> * 기본 브랜치 이름 설정: main
> ```
> git config --global init.defaultBrach main
> ```

## 3. 깃 실행
**깃 실행 방법**
> * 1. 윈도우 메뉴에서 git bash 실행
> * 2. 탐색기에서 찾은 폴더에서 'Git Bash Here'를 눌러 실행
> * 3. 탐색기에서 폴더 지정후 폴더 위치 표시 부분에 'git bash'라고 입력

**깃 명령어**
> * 리눅스 명령어
>  > * 현재 작업 디렉토리 위치 출력
>  >```
>  >$ pwd
>  >```
>  > * 파일 목록 출력
>  >```
>  >$ ls -al
>   >```
> * git 명령어 기초
>  > * 버전 확인
>  >```
>  >$ git --version
>  >```
>  > * 설정 확인
>  >```
>  >$ git config --list
>  >```
>  > * 도움
>  >```
>  >$ git --help
>  >```
