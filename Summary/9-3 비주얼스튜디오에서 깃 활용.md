# 비주얼스튜디오 코드의 깃 활용
## 1. Vscode에서 깃 활용 기초
**VSCODE 기초**
> * 폴더 열기
> * 소스 제어 활동바에서 저장소 초기화
> * 파일 생성
> * add: +
> * commit: Commit
> ```
> ## 수정 작업 취소
> $ git restore
>
> ## 옵션 --staged: index 파일 unstaged
> $ git restore --staged
> ```
> * 활동바의 Extensions에서 Git Graph설치

**현재 상태**
> * Staged Changes: 스테이지 영역
> * Changes: 작업 디렉토리

**작업 영역 수정 취소**
> * 현재의 index 파일로 수정
> ```
> $ git restore [수정할 파일]
> ```
> * Vscode: 메뉴에서 Discard Changes

**스테이지 영역 수정 취소**
> * 현재의 최근 커밋 파일로 수정
> ```
> $ git restore --staged [수정할 파일]
> ```
> * Vscode: 메뉴에서 Unstage Changes

## 2. Vscode에서 파일비교 git diff
**파일 비교**
> * 작업 디렉토리에서 원하는 하부파일 선택
> * F7를 눌로 파일비교 보기(x누르면 제거)
> * Git Graph 최상단 Uncommitted Changes를 누르고 파일 클릭
> * Git Graph의 비교할 커밋을 눌러 파일 선택
