# [실습] 지역과 원격 저장소 연동
## 1. 깃허브 원격 저장소 수정 후 지역에서 pull
**깃허브 원격 저장소 수정 후 지역에서 pull**
> * 원격 저장소에서 레포지토리 생성
>```
>$ git clone [레포지토리 URL]
>
>## 해당 레포지토리로 이동
>$ cd [저장소이름]
>
>## 브랜치 확인
>$ git branch --all
>```
> * 원격 저장소에서 생성한 레포지토리에 파일 생성
> ```
> ## 지역 저장소로 원격 저장소 업데이트 내용 반영
> $ git pull
>
> ## 원격 저장소와 지역 저장소 차이 확인
> $ git diff main origin/main
>
> ## 파일 목록 확인
> $ ls
> ```
> * 원격 저장소에서 파일 수정
> ```
> ## 원격 저장소에서 수정한 내용 끌고만 오기
> $ git fetch
>
> ## 변경 이력 확인
> $ git log --oneline --graph --all
>
> ## 원격 저장소와 지역 저장소 차이 확인
> $ git diff main origin/main
>
> ## 머지 진행
> $ git merge [머지할 브랜치]
> ```

## 2. 지역에서 원격저장소로 push
**지역에서 원격저장소로 push**
> * 지역 저장소에서 파일 생성 후 커밋
> ```
> ## git push 진행
> $ git push
> ```
> * 로그인 팝업창 발생
> * 원하는 로그인 방식 선택하여 로그인(github 계정 로그인, 토큰을 통한 로그인)
> * 반영 결과 확인
