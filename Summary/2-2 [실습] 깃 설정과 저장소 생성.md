# 깃 설정과 저장소 생성
## 1. 깃 설정
**깃을 설치한 이후 깃 배시에서 처음 할 일**
> * 사용자의 이름과 전자메일 설정
> * 주요 설정변수와 값
>  > * 기본 편집기 설정
>  >```
>  >$ git config --global core.editor ['code --wait'|'notepad']
>  >```
>  > * 줄바꿈 자동변환
>  >```
>  >$ git config --global core.autocrlf [true|false|input]
>  >```
>  > * 줄바꿈 안전확인
>  >```
>  >$ git config --global core.safecrlf[true|false]
>  >```
>  > * 사용자 이름
>  >```
>  >$ git config --globla user.name "사용자 이름"
>  >```
>  > * 사용자 전자메일
>  >```
>  >$ git config --global user.email "이메일 주소"
>  >```
> * 설정 명령과 설정 범위
>```
>$git config --설정범위 설정변수 설정값
>```
> * 설정 범위
> * --system: 모든 사용자
> * --global: 현재 사용자의 모든 저장소
> * --local: 현재 사용자의 현재 저장소

## 2. 저장소 생성
**저장소 생성**
> * 'git init' or 'git init.': 현재 디렉토리를 git repository로 만듦
> * 'git init basic': 현재 폴더 하부에 basic을 생성하고 git repository로 만듦
