# [실습] 커밋과 로그이력, 과거여행
## 1. 커밋과 로그 이력
**커밋**
>```
>## 파일 수정
>$ echo aaa > hello.txt
>$ cat hello.txt
>$ git status [--long]
>$git status -s
>
>## 파일 추가
>$ git add hello.txt
>$ git status
>$ git status [--short|-s]
>
>## 첫 번째 커밋
>$ git commit -m [커밋메세지]
>$ git status
>$ git log
>$ git log --oneline
>$ git log [--patch|-p]
>```

## 2. 과거 이전 버전으로 이동
**명령 checkout**
> * 상태가 clean해야 함
> * clean하게 하는 법
> ```
> ## 임시 저장
> $ git stash
> ```
> *과거 버전으로 이동
> ```
> $ git checkout [이전커밋]
>
> ## 다른 브랜치로 이동
> $ git branch [브랜치]
>
> ## 새로운 브랜치를 생성하고 이동
> $ git checkout -b [새로운 브랜치]
> ```
