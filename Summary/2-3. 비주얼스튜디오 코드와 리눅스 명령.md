# 비주얼 스튜디오 코드와 리눅스 명령
## 1. 비주얼 스튜디오 코드 개요
**Visual Studio Code**
> * MS사가 개발하는 오픈 소스 에디터 소프트웨어
> * 가볍고 강력한 소스 코드 편집기
> * 다양한 언어 지원

## 2. 리눅스 명령
**리눅스 명령어**
> * 현재 폴더 표시
>```
>$ pwd
>```
> * 디렉토리 변경
> ```
> $ cd
> ```
> * 디렉토리 생성
> ```
> $ mkdir
> ```
> * 파일 폴더 내용 출력
> ```
> $ ls
> ```
> * ls 다양한 옵션
>  > * -l: 파일의 상세정보
>  > * -a: 숨김 파일 표시
>  > * -al: 위 두개를 조합
> * 파일 생성
> ```
> $ touch [생성할 파일 이름]
> ```
> * 표준 출력으로 결과 표시
> ```
> $ echo git bash
> $ echo 'print()'
> ```
> * 파일 내용 보이기
> ```
> $ cat [파일 이름]
> $ cat [출력할 파일 1] [출력할 파일 2]
>
> ## 파일1과 파일2의 내용을 페이지 별로 출력
> $ cat [출력할 파일 1] [출력할 파일 2] | more
> 
> ## 파일1과 파일2의 내용을 처음부터 10번째 줄까지만 출력
> $ cat [출력할 파일 1] [출력할 파일 2] | head
>
> ## 파일1과 파일 2의 내용을 끝에서부터 10번째 줄까지만 출력
> $ cat [출력할 파일 1] [출력할 파일 2] | tail
> ```
> * 파일 a를 b로 복사
> ```
> $ cp a b
> ```
> 파일 a를 b로 이름 수정
> ```
> $ mv a b
> ```
> * 파일 및 폴더 삭제
> ```
> $ rm [삭제할 파일 이름]
> $ rm -rf [삭제할 파일 이름]
> ## -f: 강제로 파일이나 디렉토리를 삭제
> ## -r: 디렉토리 내부의 모든 내용을 삭제
> ```
