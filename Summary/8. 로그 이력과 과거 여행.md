# 로그 이력과 과거 여행
## 1. 여러 커밋과 로그 이력
**로그의 옵션**
>```
>## 문자 그림으로 로그이력 그리기
>$ git log --graph
>
>## 오래된 커밋부터 표시 --graph와 함께 사용 불가
>$ git log --reverse
>
>## 모든 브랜치의 로그 이력 표시
>$ git log --all
>
>## 최근 n개의 로그 이력 표시
>$ git log -n
>```

## 2. 과거로의 시간 여행
**현 브랜치에서 과거 HEAD로 이동**
> * 현재 브랜치에서 과거 커밋 HEAD~로 이동
>  > * HEAD: 현 커밋
>  > * HEAD~: 바로 이전 커밋
>  > * HEAD^n: n번째전 커밋
>```
>## HEAD 이전 커밋으로 이동
>$ git checkout HEAD~
>
>## 이전 checkout으로 이동
>$ git checkout -
>
>## 브랜치의 마지막 커밋으로 이동
>$ git checkout main
>```
> * 과거 커밋으로 이동은 현 커밋 상태가 깨끗해야 가능하다.
